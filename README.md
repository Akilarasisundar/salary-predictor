💰 Experience Based Salary Prediction using Machine Learning
📌 Project Overview
This project is a Salary Prediction System developed during my internship. The main objective of this project is to predict an employee's salary based on their years of experience using Machine Learning techniques.
The system analyzes the relationship between work experience and salary to build a prediction model that estimates the expected salary for a given experience level.
🎯 Objective
To develop a Machine Learning model that can predict salary based on the number of years of experience.
🛠️ Technologies Used
Python
Pandas
NumPy
Matplotlib
Scikit-learn
Flask
HTML & CSS
🤖 Machine Learning Algorithm Used
Linear Regression
⚙️ Methodology
Data Collection
Data Preprocessing
Exploratory Data Analysis (EDA)
Feature Selection
Model Training
Model Testing
Model Deployment using Flask
📊 Input Feature
Years of Experience
📈 Output
The model predicts the expected salary based on the given years of experience.
🚀 Model Workflow
User enters years of experience
Data is processed by the trained ML model
Linear Regression predicts the estimated salary
Result is displayed through a web application

Through this project, I gained practical knowledge in:
Regression-based Machine Learning

Deploying ML models into real-world applications
