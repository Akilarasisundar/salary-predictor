# 💰 Experience Based Salary Prediction using Machine Learning

## 📌 Project Overview
This project is a Salary Prediction System developed during my internship.
The main objective of this project is to predict employee salary based on years of experience using Machine Learning techniques.
The model learns the relationship between experience and salary to estimate the expected salary.
## 🎯 Objective
- Build a Machine Learning model to predict salary
- Analyze the relationship between experience and salary
- Develop a simple prediction web application
## 🛠️ Technologies Used
- Python
- Pandas
- NumPy
- Matplotlib
- Scikit-learn
- Flask
- HTML
- CSS
## 🤖 Machine Learning Algorithm
- Linear Regression
## ⚙️ Methodology
- Data Collection
- Data Preprocessing
- Exploratory Data Analysis
- Feature Selection
- Model Training
- Model Testing
- Model Deployment using Flask
## 📊 Input Feature
- Years of Experience
## 📈 Output
- Predicted Salary based on experience
## 🚀 Features
- User-friendly interface
- Real-time salary prediction
- Machine Learning based prediction
## 📚 Learning Outcome

- Improved knowledge in data analysis
- Learned ML model deployment using Flask
